HD Custom Directory Module
********************************************************************************************************
********************************************************************************************************


1. Enable Module from drush to make sure all dependencies have been met

2. This will create a page titled 'Directory' if there isn't already one

3. You must create at least one person of the content type 'Person' for this to work with the A-Z Filter

4. You can override the view by going to /admin/structure/views/view/directory/edit

5. You may have to check the public files directory for permissions if the image style isn't showing up in the popup

6. There is a JSON Feed in the view that goes into the JS file. You can add JSON Object and include them in the HTML output for the popup
